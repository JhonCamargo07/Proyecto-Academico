Más opciones

Olvidé mi contraseña

Si perdió su contraseña, por el momento la unica opción que tiene es comunicarse con el coordinador academico para que él ingrese a la base de datos y le envíe otra vez la contraseña, la cual deberá cambiar para que solo usted la conozca.

No tengo la contraseña para ingresar

Si no tiene la contraseña para poder ingresar a la página, comuniquese con el coordinador ácademico para que él le asigne una o, lo registre como un nuevo usuario en caso de que no lo tenga.

¿Cómo puedo ingresar?

Para ingresar puede dirijirse al menú que está arriba y darle clic en la opcion que diga ingresar o dando clic aquí, cabe aclarar que para ingresar debe ser un usuario valido y conocer el nombre de usuario y su contraseña, en caso de que no los conozca puede dirigirse a la opción que dice "Contraseña" allí se le dan algunas pautas que puede tomar.

Mi usuario no está activo

Si su usuario se encuentra inactivo debe comunicarse con el coordinador academico para que este le de una respuesta al respecto, es decir, él le debe decir cual es el motivo por el cual su usuario se encuentra inactivo y en caso de tratarse de un error, solucionarlo.
